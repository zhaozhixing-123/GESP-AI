# GESP.AI 产品盘点报告

> 最后更新：2026-04-13

---

## 一、项目概述

**GESP.AI** 是一个专为 **GESP（全国青少年编程等级考试）** 设计的 AI 智能学习平台。

创始人是一名五年级小学生，备考 GESP 时觉得私教太贵，于是用 AI 技术自建了这个平台，并用它成功通过了 GESP 五级认证——成为产品的第一个验证用户。

**核心定位**：真题刷题 x AI 引导思考 x 变形题巩固 —— 从「做错」到「真懂」的完整闭环。

**技术栈**：

| 层级 | 技术选型 |
|------|----------|
| 前端框架 | Next.js 16 + React 19 |
| 样式 | Tailwind CSS 4 + @tailwindcss/typography |
| 代码编辑器 | Monaco Editor（VS Code 同款） |
| 数据库 | PostgreSQL + Prisma 7 ORM |
| AI 引擎 | Claude API（Opus 4.6 / Sonnet 4.6） |
| 在线评测 | Judge0 CE（RapidAPI） |
| 支付 | 虎皮椒（微信扫码） |
| Markdown 渲染 | react-markdown + remark-gfm + remark-math + rehype-katex |
| 认证 | JWT + bcryptjs |

**数据规模**：200+ 道历年真题，800+ 道 AI 变形题，覆盖 1-8 级全部级别。

## 二、现有功能清单

### 2.1 题库系统

- 200+ 道 GESP 历年真题，覆盖 1-8 级
- 按级别筛选（1-8 级）、关键词搜索（题号 / 题名）
- 分页浏览，每道题标注知识点标签和级别
- 题目数据从**洛谷自动爬取**，支持单题导入和批量导入（题单页 / 列表页）
- **AI 自动打知识点标签**（管理后台触发，分类如递推/递归、动态规划等）
- 题目状态标记：已通过（绿色 ✓）、尝试过（黄色圆点）、未做

### 2.2 在线编程 & 评测系统

- **Monaco 代码编辑器**：C++ 语法高亮、暗色主题、自动补全、代码选中
- **Judge0 云端评测**：代码在远程沙箱编译执行，支持六种判题状态：
  - AC（通过）、WA（答案错误）、TLE（超时）、CE（编译错误）、RE（运行错误）、MLE（内存超限）
- **运行样例**：一键运行所有样例查看结果
- **自定义输入运行**：学生可以自己构造测试数据调试
- **正式提交判题**：用隐藏测试数据做完整判定
- **批量判题优化**：所有测试点并行提交 Judge0，显著提升评测速度
- 提交记录查看（时间、内存、状态、时间戳）
- 左右分栏布局，**可拖拽调整比例**

### 2.3 AI 老师（核心功能）

- 由 **Claude Opus 4.6** 驱动的苏格拉底式 AI 编程教师
- **绝不直接给答案**，只用引导式提问帮学生自己想明白
- 支持**流式输出**（SSE），实时看到 AI 逐字输出过程
- **聊天历史持久化**到数据库，再次打开同一题目继续对话
- 可选「附带当前代码」让 AI 看到学生的代码上下文
- **选中代码片段 → 一键「解释选中代码」**
- AC 后自动提示「让 AI 点评代码」（分析时间复杂度、代码风格、更优解）
- 免费用户每题限 5 次对话，付费用户无限制
- 系统提示词（System Prompt）支持管理员在后台在线编辑

### 2.4 变形题系统（独家）

- 每道真题配套 **AI 生成的变形题**（共 800+ 道）
- **做错才解锁**机制：
  - 真题 WA → 解锁第一批变形题（batch 1）
  - 第一批变形题也 WA → 解锁第二批变形题（batch 2）
- AI 出题流程：
  1. Claude Sonnet 4.6 通过 **tool_use** 强制结构化生成变形题题面
  2. AI 自动写 C++ 解法
  3. Judge0 运行解法计算样例输出
- 变形题保持相同算法思路和知识点，但换情境、换背景、换参数
- Sonnet 失败 3 次自动 **fallback 到 Opus** 兜底
- 去重机制：已有变形题标题传给 AI 避免雷同
- 变形题在题库中用琥珀色标注，与真题视觉区分

### 2.5 错题本 + AI 错因分析

- 做错自动收录 / 手动加入错题本
- **AI 错因分析**（Claude Opus 4.6 流式输出）：
  - 自动识别 **13 种错误类型**：数组越界、逻辑错误、边界条件、整数溢出、死循环、输入输出错误、算法错误、变量未初始化、递归错误、语法错误、超时优化、内存超限、其他
  - 结构化三段式输出：
    1. **这道题错在哪** —— 具体指出代码错误
    2. **这类错误为什么容易犯** —— 脱离本题讲通用规律，附无关示例
    3. **下次怎么避免** —— 给出自检习惯清单
- **薄弱知识点雷达**：统计待解决题目中各知识点频次，可视化排序展示
- **最常犯错误趋势**：汇总错误类型分布（如「逻辑错误 x3、边界条件 x2」）
- 支持多维筛选：按状态（待解决 / 已掌握）、级别（1-8 级）、知识点标签
- 分析结果持久化到数据库，避免重复消耗 AI 调用

### 2.6 模拟考试

- 仿 GESP 真实考试环境
- 可配置项：
  - 考试级别：1-8 级
  - 题目数量：3 / 5 / 10 道
  - 时间限制：20 / 40 / 60 分钟
- **倒计时器**（最后 5 分钟变橙，最后 1 分钟变红）
- 考试期间**无 AI 辅助**，独立作答
- 题目导航栏（已写代码 / 样例已AC / 未写，用不同颜色标记）
- 考试结束后自动执行：
  1. 批量运行所有样例，统计通过情况
  2. **AI 生成诊断报告**（Claude Opus 4.6 流式输出）：
     - 整体表现评价
     - 逐题点评（思路、问题、改进建议）
     - 知识盲区诊断（归纳 2-3 个需加强的知识点）
     - 下一步学习计划（3 条具体建议）

### 2.7 专注力追踪 & 家长监控

- **FocusTracker** 实时计时：专注时间 vs 分心时间，顶栏实时显示
- 分心检测机制：
  - 页面失焦（切换标签页 / 切到其他应用）
  - 窗口大幅缩小（宽度减少 > 200px）
- 防作弊措施：
  - 禁用右键菜单
  - 禁用 F12 / Ctrl+Shift+I/J/C（DevTools 快捷键）
- **飞书 Webhook 通知**：分心累计满 2 分钟，自动给家长发送飞书机器人消息
- 数据 localStorage 持久化，跨天自动重置
- **独立家长密码保护**：需要单独设置和验证家长密码才能进入设置页，防止孩子自行修改
- 支持 Webhook 测试消息发送

### 2.8 AI 自动生成测试数据（管理后台）

- 自动化流程：
  1. AI 编写**两个独立解法**（暴力法 + 不同思路）
  2. 先用样例**交叉验证**两个解法的正确性
  3. AI 生成 15 组测试输入（边界值、特殊情况、随机中等规模）
  4. 两个解法分别运行所有测试输入
  5. **只保留两个解法输出一致的测试点**，自动丢弃不一致的
- 模型策略：Sonnet 4.6 尝试 3 次 → Opus 4.6 fallback 1 次

### 2.9 用户系统 & 订阅支付

- 注册 / 登录（JWT 鉴权 + bcrypt 密码加密）
- **Freemium 模式**：
  - 免费体验 1 道完整题目（含 AI 对话 5 次、变形题解锁、错因分析 1 次）
  - 付费解锁全部功能
- 三档付费套餐：

| 套餐 | 价格 | 折合月费 | 定位 |
|------|------|----------|------|
| 月卡 | ¥99/月 | ¥99 | 灵活体验 |
| 季卡 | ¥199/季 | ~¥66 | 最受欢迎（推荐） |
| 年卡 | ¥599/年 | ~¥49 | 长期备考，最划算 |

- **虎皮椒微信扫码支付**：生成二维码 → 前端轮询支付状态 → 自动跳转成功页
- 续费自动延期（从当前到期日往后算，已过期则从当天算）
- 顶栏动态展示会员剩余天数，≤3 天红色提醒，≤7 天橙色提醒

### 2.10 个人中心

- 查看账户信息（用户名、注册时间、提交次数）
- 订阅状态 & 到期时间展示
- 可编辑学习信息：目标级别（3-8 级）、考试日期、手机号

### 2.11 管理后台

- **题目管理**：增删改查、从洛谷导入（单题/批量）、AI 打标签、清除标签/题目
- **提示词管理**：在线编辑 AI 老师 System Prompt 和错题分析 Prompt，支持 `{{variable}}` 模板变量
- **用户管理**：查看和管理所有注册用户
- **变形题管理**：单题或批量触发 AI 生成变形题，查看生成状态和错误
- **测试数据生成 & 验证**：为题目一键生成测试数据并验证

## 三、技术亮点

### 3.1 AI 全链路深度集成

全平台共有 **5 个 AI 场景**，覆盖教学全流程：

| AI 场景 | 模型 | 用途 |
|---------|------|------|
| AI 老师对话 | Claude Opus 4.6 | 苏格拉底式引导教学 |
| 错因分析 | Claude Opus 4.6 | 结构化错误诊断 + 举一反三 |
| 变形题生成 | Claude Sonnet 4.6 (+ Opus fallback) | 同知识点换情境出题 |
| 测试数据生成 | Claude Sonnet 4.6 (+ Opus fallback) | 双解法交叉验证生成测试点 |
| 知识点标注 | Claude API | 自动为题目打知识点标签 |

### 3.2 智能模型选择策略

- **教学和分析场景**用 Opus 4.6（质量优先）
- **出题和生成场景**用 Sonnet 4.6（性价比优先）+ Opus fallback（兜底保障）
- 失败重试机制：Sonnet 最多 3 次 → 切换 Opus 1 次

### 3.3 tool_use 强制结构化输出

变形题和测试数据生成使用 Claude **tool_use + tool_choice** 强制调用指定工具，确保输出为严格的结构化数据，避免 JSON 解析失败或格式不一致。

### 3.4 双解法交叉验证

测试数据生成时，AI 编写两个完全独立的解法（暴力法 + 不同思路），分别运行所有输入，**只保留两个解法输出完全一致的测试点**。这种交叉验证机制大幅提升了测试数据的可靠性。

### 3.5 全流式交互体验

AI 对话、错因分析、模考诊断报告全部采用 **SSE（Server-Sent Events）流式输出**，用户可以实时看到 AI 逐字生成的过程，响应感极强，无需等待完整生成。

### 3.6 Judge0 批量评测优化

多测试点通过 `submissions/batch` 接口**并行提交**，配合批量查询接口轮询结果，比逐个串行提交大幅提升评测速度。

### 3.7 提示词热更新

AI 老师的 System Prompt 和错题分析 Prompt 存储在数据库 `Prompt` 表中，管理员可在后台在线编辑，支持 `{{problem_title}}`、`{{user_code_section}}` 等模板变量，**无需改代码即可调整 AI 行为**。

### 3.8 Freemium 精细限流

三个维度独立控制免费额度：

| 维度 | 免费额度 | 付费 |
|------|----------|------|
| 题目数 | 1 道 | 全部 |
| AI 对话 | 每题 5 次 | 无限 |
| 错因分析 | 1 次 | 无限 |

## 四、用户价值分析

### 4.1 解决的核心痛点

| 痛点 | GESP.AI 的解法 |
|------|----------------|
| **私教太贵**（¥200-500/小时，月费 ¥2000+） | AI 老师 24 小时在线，月费仅 ¥99，省 97% |
| **背题不管用，换情境就不会** | 变形题系统：做错解锁同知识点变形题，换情境再练，确认真懂 |
| **不知道自己的盲点在哪** | 错题本薄弱知识点雷达 + 错误类型趋势分析，精准定位薄弱环节 |
| **做对了不确定是真懂还是蒙的** | 做错解锁变形题机制，通过举一反三验证真实理解程度 |
| **家长无法监控学习状态** | 专注力追踪 + 飞书通知，分心 2 分钟自动提醒家长 |
| **缺乏模考环境，不适应考试节奏** | 仿真模拟考试 + AI 诊断报告，提前适应 GESP 考场环境 |
| **AI 工具直接给答案，养成依赖** | 苏格拉底式引导教学，绝不给答案，培养独立思考能力 |
| **错题不知道怎么有效复盘** | AI 自动分析错因 + 给出通用自检习惯，帮助举一反三、避免重复犯错 |

### 4.2 核心价值闭环

```
做真题 → 做错了 → AI 分析错因 → 解锁变形题 → 换情境再练 → 真正掌握
   ↑                                                          |
   └──────────── 错题本 + 薄弱知识点 → 精准刷题 ←─────────────┘
```

将学生从「做错 → 不知道为什么错 → 下次还错」的死循环，转变为 **「做错 → AI 分析错因 → 变形题验证 → 真正掌握」** 的正向学习闭环。

### 4.3 差异化竞争优势

1. **变形题系统是独家能力**——市面上其他 OJ 平台只有固定题库，没有 AI 按知识点动态生成变形题
2. **苏格拉底式 AI 教学**——不同于 ChatGPT 直接给答案，刻意设计为「引导而不告知」
3. **创始人即用户**——产品由一名备考中的小学生自己设计和验证，天然贴合目标用户需求
4. **家长友好**——专注力监控 + 飞书通知解决了家长「看不见孩子学习状态」的焦虑
